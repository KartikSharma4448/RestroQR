# UI Package

This package will contain all Jetpack Compose screens and components:

- `auth/` — Login and Registration screens
- `dashboard/` — Main dashboard screen
- `profile/` — Restaurant profile setup screen
- `menu/` — Categories and food items management screens
- `qr/` — QR code display and download screen

Each screen will follow the MVVM pattern with Hilt-injected ViewModels.
